# 🧠 The Skills Matrix

Welcome to the `skills/` directory. This folder serves as the theoretical foundation and competency matrix for the **Modern Web Craft** monorepo. 

While the `packages/` and `apps/` directories contain the *execution* of these concepts, the documents in this folder define the *standards* we adhere to. They outline what it means to build Production-Grade UI and advanced web architecture.

## 📂 Directory Index

* **[01-frontend-design.md](./01-frontend-design.md)** - Core UI, Accessibility (a11y), and Component Architecture.
* **[02-design-to-code.md](./02-design-to-code.md)** - Design Tokens, Headless UI, and Styling Systems (`@craft/ui-system`).
* **[03-animation-interactivity.md](./03-animation-interactivity.md)** - Performance-first Motion and Micro-interactions (`@craft/motion`).
* **[04-performance-optimization.md](./04-performance-optimization.md)** - Core Web Vitals, Rendering Patterns, and Asset Delivery.
* **[05-seo-content.md](./05-seo-content.md)** - Structured Data, Technical SEO, and Headless Content Integration (`@craft/seo-utils`, `@craft/content-engine`).
* **[06-browser-automation.md](./06-browser-automation.md)** - E2E Testing, Visual Regression, and CI/CD Guardrails (`tools/browser-automation`).
* **[07-generative-ui.md](./07-generative-ui.md)** - AI Context Compiling, LLM Integration, and Structured UI Output (`@craft/gen-ui`).

*Use these files as a roadmap for personal growth, a rubric for code reviews, or a discussion guide for technical interviews.*
